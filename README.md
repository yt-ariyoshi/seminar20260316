## データセットについて

本リポジトリでは，miniJSRT_database を用いたデモコードを公開していますが，
データセット本体は同梱していません。

miniJSRT_database は，日本放射線技術学会 画像部会が配布しているデータセットです。
データセットの入手は，必ず公式配布元から各自で行ってください。

- miniJSRT_database:
  http://imgcom.jsrt.or.jp/minijsrtdb/

## 重要
